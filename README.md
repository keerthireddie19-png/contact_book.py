# Contact Book (Python)

This is a simple Contact Book application built using Python. It allows users to store and manage contact details easily.

## Features
- Add a new contact
- View all contacts
- Search for a contact
- Delete a contact

## Technologies Used
- Python

## How to Run
1. Install Python on your system.
2. Download or clone this repository.
3. Open the project folder.
4. Run the file:

```bash
python contacts_book.py
